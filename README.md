# React 前端工程脚手架

- 支持 SCSS
- 支持 React hot loader
- babel 7
- webpack 4

## 使用

先安装 babel cli, 确保 babel 版本为 7

```shell
npm i @babel/core @babel/node @babel/cli -g

babel -v # -> 7
babel-node -v # -> 7
```

```shell
git clone https://github.com/SANGET/react-app-seed.git yourProjectName
cd yourProjectName
npm run init
```
